### 1. 목표와 대상

- Spring의 핵심적인 기술에 대해서 서술하되, 자신의 공부가 아닌 스프링 입문자를 위해 정리
- 기술 정리는 레이아웃에 맞춰 서술

### 2. 담당 역할

- 준우 : AOP
- 혜성 : 빈 생성 원리
- 상훈 : Dispatcher Servlet
